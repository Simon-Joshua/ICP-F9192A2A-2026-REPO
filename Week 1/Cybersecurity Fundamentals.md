##Cybersecurity Fundamentals

The fundamentals of cybersecurity are the core principles and practices used to protect systems, networks, applications, and data from unauthorized access, attacks, and damage.

**1. The CIA Triad**
	This is the foundation of cybersecurity:
	a) Confidentiality - Only authorized people can access information.
	b) Integrity - Data remains accurate and unaltered.
	c) Availability - Systems and data are accessible when needed.

These three principles guide most security decisions.

**2. Authentication and Authorization**
**Authentication**
	Verifying who someone is.

	Examples: Passwords, Biometrics (Fingerprints, face recognition), Multi-factor Authentication.

**Authorization**
	Determining what an authenticated user is allowed to do.

	Examples: Admin permissions, Read-only access, Role-based access control.

**3. Networking Fundamentals**
The understanding of the following is essential as many attacks exploit network communications;

- IP addresses
- Subnets
- Routers and switches
- DNS
- HTTP/HTTPS
- TCP and UDP
- Firewalls
- VPNs

**4. Security Principles**
**Least Privilege**
Users should only have the permissions necessary to perform their tasks.

**Defense in Depth**
Use multiple layers of protection instead of relying on one control.

Example:

- Firewall
- Antivirus
- MFA
- Encryption

**Zero Trust** 
Never automatically trust a user or device, even if it's inside the network.

**5. Common Threats & Vulnerability Management**
These are a list of common threats to systems and networks;

- Phishing
- Malware
- Ransomware
- Social engineering
- Password attacks
- Denial-of-Service (DoS)
- Man-in-the-Middle attacks
- Insider threats

Understanding how attacks work helps you defend against them.

**Vulnerability Management**
Organizations continuously:

1. Identify vulnerabilities.
2. Assess risk.
3. Apply patches.
4. Verify fixes.

Common tools include vulnerability scanners and security monitoring platforms.
